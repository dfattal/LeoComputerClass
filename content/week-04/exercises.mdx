You will need your gates from Week 1, `get_bit` from Week 2, and `half_adder` from Week 3. Start your file with:

```python
def AND(a, b):
    return a & b

def OR(a, b):
    return a | b

def NOT(a):
    return 1 - a

def XOR(a, b):
    return OR(AND(a, NOT(b)), AND(NOT(a), b))

def get_bit(x, i):
    return (x >> i) & 1

def half_adder(a, b):
    s = XOR(a, b)
    c = AND(a, b)
    return (s, c)
```
