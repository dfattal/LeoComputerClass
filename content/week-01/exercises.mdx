## Warm-Up: How Functions Work

Before we jump into gates, let's make sure you're comfortable writing and testing a Python function.

Here's a silly example — a function that always returns 1, no matter what:

```python
def always_one(a, b):
    return 1

# Test it:
print(always_one(0, 0))  # → 1
print(always_one(0, 1))  # → 1
print(always_one(1, 0))  # → 1
print(always_one(1, 1))  # → 1
```

See the pattern? You **define** a function with `def`, you **return** a value, and you **test** it with `print()`. You'll do exactly this for each gate below.
