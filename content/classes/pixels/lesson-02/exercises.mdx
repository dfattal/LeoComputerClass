### Exercise 1 — `my_square()`

Make a box called `c` that holds `"purple"`. Then give back a **2x2 square** (two rows, two squares each) that uses `c` for all four squares.

```python
my_square()   # -> [["purple", "purple"], ["purple", "purple"]]
```

**Hint:** first line inside the function: `c = "purple"`. Then `return [[c, c], [c, c]]`. Notice you never typed the word "purple" inside the grid — only the box name `c`.

**Then play:** change `"purple"` to `"green"` and press Run. All four squares change from one little edit!

### Exercise 2 — `two_tone()`

Make **two** boxes — `a = "black"` and `b = "yellow"` — and give back **one row** that goes `a`, `b`, `a`, `b`.

```python
two_tone()   # -> [["black", "yellow", "black", "yellow"]]
```

**Hint:** `return [[a, b, a, b]]`. The boxes `a` and `b` each get used twice — that's the whole point of giving them names.
