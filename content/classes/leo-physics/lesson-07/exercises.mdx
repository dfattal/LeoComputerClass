```python
        s = x0 - d

    return v
```

**Examples:**
- `muzzle_speed(0.1, 200, 0.02, 0.3, 0.0001)` → about `10.0`
- `muzzle_speed(0.12, 300, 0.02, 0.3, 0.0001)` → about `14.7`
- `muzzle_speed(0.1, 200, 0.05, 0.3, 0.0001)` → about `6.32` (heavier dart → slower)

**Hint:** `v = v + (F/m)*dt`, then `d = d + v*dt`. The acceleration is the spring force divided by mass.

## Tips If You're Stuck

1. **`blast_range` errors about `drag_flight`?** Make sure you didn't delete the provided `drag_flight` function in the editor.
2. **Everything hangs / times out?** Your `muzzle_speed` loop must update `d` (`d = d + v*dt`) so the dart reaches the end of the barrel.
3. **Range is 0 or `None`?** Check that `blast_range` actually `return`s the result of `drag_flight(...)[0]`.
4. **`blast_report` list looks wrong?** It's `[speed, flight[0], flight[1]]` — speed first, then range, then peak.
