## Exercise 2: Muzzle Velocity

Write `muzzle_velocity(x0, k, m, barrel, dt)` that simulates the dart from rest and returns how fast it's going when it leaves the barrel.

- `x0` ‚Äî how far the spring starts squished (meters)
- `k` ‚Äî spring stiffness, `m` ‚Äî dart mass
- `barrel` ‚Äî barrel length (meters); the dart exits when it has travelled this far
- `dt` ‚Äî time step

The dart's distance travelled is `d`. The spring's remaining squish is `s = x0 - d`. Use your `spring_force` each step.


- `muzzle_velocity(0.1, 200, 0.05, 0.3, 0.0001)` ‚Üí about `6.32` (matches `0.1 * sqrt(200/0.05)`!)

- `muzzle_velocity(0.2, 200, 0.05, 0.1, 0.0001)` ‚Üí about `10.95` (short barrel ‚Äî the spring isn't done pushing yet)

**Hint:** Inside the loop: `v = v + a * dt`, then `d = d + v * dt`. The acceleration starts big and fades to zero as `s` shrinks ‚Äî that's the spring uncoiling. This reuses your `spring_force` from Exercise 1!

## Tips If You're Stuck

1. **`muzzle_velocity` runs forever (timeout)?** Make sure you update `d` inside the loop (`d = d + v * dt`) so the dart actually moves down the barrel.
2. **Muzzle velocity is 0?** Check the order: update `v` first (`v = v + a * dt`), *then* `d`. If you move `d` before `v`, the first step has no speed.
3. **Exercise 3 way off?** It's `x0 * sqrt(k/m)`, not `sqrt(x0 * k/m)` ‚Äî the `x0` is outside the square root.
4. **Spring "pulls back"?** Your `spring_force` must return `0` once `compression <= 0`.
