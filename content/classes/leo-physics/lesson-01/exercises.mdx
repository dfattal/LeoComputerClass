## Tips If You're Stuck

1. **Answer way off (like it's going *up*)?** Check the sign on gravity — it should be `-g`, not `+g`.
2. **Answer a little off but the test fails?** Make sure you update velocity *first*, then position, inside the loop.
3. **`None` returned?** Don't forget `return h` at the end (and `return` in Exercises 1 and 2).
4. **Still stuck?** Re-read "Putting It in a Loop" in the lesson — the falling-ball code there is almost exactly Exercise 3.
