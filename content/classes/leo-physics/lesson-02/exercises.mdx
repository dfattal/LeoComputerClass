## Tips If You're Stuck

1. **Dart flies the wrong way / weird numbers?** You probably forgot `math.radians`. Python trig needs radians, not degrees.
2. **Loop never stops (timeout)?** Make sure `y` actually comes back down ‚Äî check that gravity is `-g` (negative) and that you have an `if ... : break`.
3. **Range a little off?** Double-check the update order: `vy` first, then `x` and `y`. And make sure `vx` has *no* update line.
4. **Returned `None`?** Don't forget the `return` at the end.
