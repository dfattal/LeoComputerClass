## Only the outside matters

Remember how electrons stack in shells — **2** in the first, then **8**, then **8**? It turns out an atom mostly cares about just **one** of those shells: the **outermost** one. The electrons way out on the edge are the ones that bump into other atoms, so they decide how the atom behaves.

We call that count the atom's **family number** (scientists call it the *valence*).

```python
shells(11)            # -> [2, 8, 1]   sodium
outer_electrons(11)   # -> 1           just 1 lonely electron on the outside
```

Sodium has 11 electrons, but only **1** is in the outer shell. That single outer electron is its whole personality.

## Same outside = same family

Here's the cool part. **Lithium** has 3 electrons → `[2, 1]` → **1** outer. **Sodium** has 11 → `[2, 8, 1]` → also **1** outer. Different atoms, but the *same* number of outer electrons — so they act like cousins. They're in the same **family**.

```python
same_family(3, 11)   # -> True   lithium & sodium: both 1 outer
same_family(6, 8)    # -> False  carbon (4 outer) & oxygen (6 outer)
```

## Why the table has columns

If you line up all the elements in order and start a new row every time a shell fills up, something magical happens: the families stack into **columns**. Every element in a column has the same number of outer electrons.

That grid is the famous **periodic table** — and you're about to draw it yourself, colored by family. 🌈
