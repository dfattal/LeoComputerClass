## The pH scale

Every liquid gets a number called its **pH**, from **0 to 14**:

- **0 to 6** → **acid** (sour stuff: lemon 🍋, vinegar, soda)
- **7** → **neutral** (pure water 💧 — right in the middle)
- **8 to 14** → **base** (slippery stuff: soap 🧼, baking soda, bleach)

The further from 7, the *stronger* it is. Lemon (pH 2) is a strong acid; bleach (pH 13) is a strong base.

## Cabbage juice: the color spy

Here's the cool trick: **red-cabbage juice** turns a different color at each pH. Drop it in and watch:

| pH | What it is | Color |
|----|-----------|-------|
| 0–3 | strong acid | 🔴 red |
| 4–6 | mild acid | 🟣 purple |
| 7 | neutral | 🔵 blue |
| 8–11 | mild base | 🟢 green |
| 12–14 | strong base | 🟡 yellow |

So a flash of red means "strong acid!" and yellow means "strong base!" One drop tells you everything.

```python
cabbage_color(2)   # -> "red"     lemon juice
cabbage_color(7)   # -> "blue"    pure water
cabbage_color(13)  # -> "yellow"  bleach
is_acid(2)         # -> True
is_acid(10)        # -> False     that's a base
```
