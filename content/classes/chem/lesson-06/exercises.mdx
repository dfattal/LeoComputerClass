### Exercise 1 — `cabbage_color(pH)`

Return the color red-cabbage juice turns at a given pH.

```python
cabbage_color(0)    # -> "red"
cabbage_color(5)    # -> "purple"
cabbage_color(7)    # -> "blue"
cabbage_color(9)    # -> "green"
cabbage_color(14)   # -> "yellow"
```

The rule:
- pH **0–3** → `"red"`
- pH **4–6** → `"purple"`
- pH **7** → `"blue"`
- pH **8–11** → `"green"`
- pH **12–14** → `"yellow"`

**Hint:** check from low to high with `if` / `elif`. Once you've ruled out 0–3, `elif pH <= 6` neatly catches 4, 5, and 6.
