### Exercise 1 — `bonds_needed(outer, wants)`

How many bonds does an atom make? Exactly how many more electrons it needs to fill its outer shell.

```python
bonds_needed(1, 2)   # -> 1   hydrogen
bonds_needed(6, 8)   # -> 2   oxygen (a double bond!)
bonds_needed(4, 8)   # -> 4   carbon
bonds_needed(7, 8)   # -> 1   chlorine
```

**Hint:** it's a subtraction — `return wants - outer`.

### Exercise 2 — `molecule_atoms(counts)`

Add up all the atoms in a molecule. `counts` is a list telling you how many of each kind.

```python
molecule_atoms([2, 1])   # -> 3   water (2 H + 1 O)
molecule_atoms([2])      # -> 2   oxygen gas (2 O)
molecule_atoms([1, 4])   # -> 5   methane (1 C + 4 H)
```

**Hint:** Python has a built-in for this — `return sum(counts)`.
