You'll build a **codebreaker** that finds a Caesar key without being told it. The starter already gives you `shift_letter` and `caesar_decrypt` from Lesson 1, plus a list called `COMMON_WORDS` (the most common little words in English). Use them!

## Tips If You're Stuck

1. **`all_shifts` not 26 long?** `range(26)` gives 0 through 25 — that's 26 values. 
2. **Score always 0?** Make sure you're splitting into words (`text.split()`) and checking `in COMMON_SET`, not checking letters.
3. **Cracker returns the wrong shift?** Start `best_score` at `-1` so the very first shift can win, and only update when the new score is strictly greater.
4. **Want to see the answer?** After cracking, print `caesar_decrypt(message, crack_caesar(message))` to read the secret out loud.

Crack all three and you're officially a codebreaker. Next: a cipher with so many keys that brute force gives up. Let's go!
