# Lesson 1 — Exercises: The Caesar Cipher

## How This Works

You'll write three functions that together make a **Caesar cipher machine**. When you click **Run Tests**, the computer calls your functions with secret messages and checks that they scramble and unscramble correctly.

**The rules every function follows:**

- Work on lowercase letters **`a`–`z`** only.
- **Spaces and other characters pass through unchanged** (so `attack at dawn` keeps its spaces).
- The alphabet **wraps around**: after `z` comes `a`. Use `% 26` to wrap.

Remember the one-line trick for shifting a single letter:

```python
chr((ord(letter) - 97 + n) % 26 + 97)
```

## Tips If You're Stuck

1. **Letters running off the end (like `z` → `{`)?** You forgot the `% 26`. That's what loops the ring back to `a`.
2. **Spaces turned into weird symbols?** Make sure non-letters return unchanged — check `if "a" <= letter <= "z":` before shifting.
3. **Decrypt not getting back the original?** You're probably shifting the same direction as encrypt. Decrypt shifts by `-n`, not `n`.
4. **`None` showing up?** Don't forget `return` in each function.
