# Lesson 5 ‚Äî Exercises: The XOR Trick

## How This Works

In Python, XOR is the `^` operator. Your key is a single number from `0` to `255`. To XOR a letter you first turn it into a number with `ord(c)`, and to turn a number back into a letter you use `chr(n)`.

The big idea to lean on: **XOR with the same key twice returns the original.** That's why `xor_encrypt` and `xor_decrypt` use the exact same XOR ‚Äî just in opposite directions (letters‚Üínumbers, then numbers‚Üíletters).

## Tips If You're Stuck

1. **Got letters instead of a list from encrypt?** `xor_encrypt` returns **numbers**, not a string. Don't call `chr` there ‚Äî that's the decrypt step's job.
2. **Decrypt gives weird symbols?** Make sure you XOR with the **same** key you encrypted with. A different key won't undo it.
3. **`TypeError` with ord/chr?** `ord` takes a one-character string and gives a number; `chr` takes a number and gives a one-character string. Don't mix them up.
4. **Want proof?** Print `xor_decrypt(xor_encrypt("top secret", 200), 200)` ‚Äî it should say `top secret`.
