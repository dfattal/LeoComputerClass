# Strings Are Superpowers

## Strings Are Like Necklaces

Remember strings from last lesson? A string is just a bunch of characters all strung together -- like beads on a necklace.

```python
message = "hello spy"
```

Each bead (character) has a **position number** starting from **0**:

```
 h   e   l   l   o       s   p   y
[0] [1] [2] [3] [4] [5] [6] [7] [8]
```

Today you're going to learn the **superpowers** that every string comes with. Think of yourself as a **spy** -- and these superpowers are your tools for decoding secret messages!

## Superpower 1: Grabbing a Single Character (Indexing)

You can grab any bead from the necklace using square brackets:

```python
text = "hello spy"
print(text[0])   # "h" -- the first character
print(text[6])   # "s"
```

Want the **last** character without counting? Use a negative number:

```python
print(text[-1])  # "y" -- last character
print(text[-2])  # "p" -- second to last
```

Negative numbers count backwards from the end. Handy!

## Superpower 2: Slicing -- Cutting Out a Piece

Slicing lets you grab a **chunk** of the string. You write `text[start:stop]` and it gives you everything from `start` up to (but **not including**) `stop`.

```python
text = "hello spy"
print(text[0:5])   # "hello"
print(text[6:9])   # "spy"
```

There are some shortcuts too:

```python
print(text[:5])    # "hello" -- from the beginning to position 5
print(text[6:])    # "spy"   -- from position 6 to the end
```

## Superpower 3: The Backwards Trick

Here's a **magic trick** every spy needs -- reversing a message!

```python
secret = "yps olleh"
print(secret[::-1])  # "hello spy"
```

`[::-1]` means "go through the whole string, but **step backwards**." It flips the entire string around. This is one of the coolest Python tricks you'll learn!

## Superpower 4: String Methods

Every string in Python comes with built-in **methods** -- think of them as superpowers glued right onto the string. You use them with a dot:

### `.upper()` and `.lower()` -- Change the Case

```python
print("whisper".upper())   # "WHISPER"
print("SHOUT".lower())     # "shout"
```

### `.count("x")` -- Count How Many Times Something Appears

```python
print("banana".count("a"))   # 3
print("hello".count("l"))    # 2
```

### `.replace("old", "new")` -- Swap Out Text

```python
print("cats are cool".replace("cats", "dogs"))  # "dogs are cool"
```

It replaces **every** match it finds, not just the first one.

### `.find("x")` -- Where Does It First Appear?

```python
print("hello spy".find("spy"))  # 6
print("hello spy".find("xyz"))  # -1 (not found)
```

## Superpower 5: `len()` -- How Long Is It?

`len()` tells you how many characters are in a string (including spaces!):

```python
print(len("hello"))       # 5
print(len("hello spy"))   # 9
```

## Superpower 6: Building Strings with `+` and `*`

You can **glue** strings together with `+`:

```python
print("secret" + " " + "code")  # "secret code"
```

And **repeat** them with `*`:

```python
print("ha" * 3)    # "hahaha"
print("*" * 5)     # "*****"
```

This is super useful when you need to build a string of a certain length!

## Putting It All Together

Now you have a full spy toolkit:

| Superpower | What It Does | Example |
|---|---|---|
| `text[0]` | Get one character | `"spy"[0]` → `"s"` |
| `text[1:4]` | Slice out a chunk | `"hello"[1:4]` → `"ell"` |
| `text[::-1]` | Reverse the string | `"yps"[::-1]` → `"spy"` |
| `.count("x")` | Count occurrences | `"banana".count("a")` → `3` |
| `.replace(a, b)` | Swap text | `"hi".replace("hi","bye")` → `"bye"` |
| `.find("x")` | Find position | `"spy".find("p")` → `1` |
| `.lower()` / `.upper()` | Change case | `"HI".lower()` → `"hi"` |
| `len(text)` | String length | `len("spy")` → `3` |
| `+` | Glue strings | `"a" + "b"` → `"ab"` |
| `*` | Repeat strings | `"ha" * 2` → `"haha"` |

Time to use these superpowers to crack some codes!
