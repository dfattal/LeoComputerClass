# Exercise: Reverse Complement

## Your Task

Write a function called `reverse_complement(dna)` that returns the **reverse complement** of a DNA string.

### The Two Steps

1. **Complement** -- swap each base for its partner (A/T, C/G)
2. **Reverse** -- flip the string backwards

## Example

```python
reverse_complement("ATCG")
# Step 1 - complement: "TAGC"
# Step 2 - reverse:    "CGAT"
# Returns "CGAT"

reverse_complement("AAAA")     # Returns "TTTT"
reverse_complement("GCTA")     # Returns "TAGC"
```
