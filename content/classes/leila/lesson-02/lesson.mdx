# Complementary DNA

## DNA Has Two Strands!

Remember how DNA is written in four letters -- A, T, C, and G? Well, here's something amazing: DNA isn't just one strand of letters. It's actually **two strands** stuck together, like a **zipper**!

## Base Pairing: The Buddy System

Each base on one strand has a specific **partner** on the other strand. They always pair up the same way:

- **A** always pairs with **T**
- **T** always pairs with **A**
- **C** always pairs with **G**
- **G** always pairs with **C**

Think of it like a zipper where each tooth only fits with **one specific** tooth on the other side. A and T are best friends. C and G are best friends. They always hold hands!

```
Strand 1:  A  T  C  G  A  T
           |  |  |  |  |  |
Strand 2:  T  A  G  C  T  A
```

## Why Does This Matter?

This buddy system is how DNA **copies itself**! When a cell needs to divide, it unzips the DNA down the middle. Then each strand builds a new partner by following the pairing rules. That's how every cell in your body gets a perfect copy of your DNA.

It's like having half a zipper and knowing exactly which teeth to add to make it whole again!

## Fun Fact

If you stretched out all the DNA in just one of your cells, it would be about **2 meters long** (taller than most adults!). And your body has about 37 trillion cells. That's a LOT of zippers!
