# Lesson 6 — Exercises: The Lunar Lander

## How This Works

Three functions to land on the Moon: one step, the full descent, and the safety check. The Moon's gravity is `g = 1.62`.

**Remember:** up is positive, so a **falling** lander has a **negative** velocity. The touchdown speed comes out negative — that's fine, `is_safe` uses `abs()`.

**Important:** This is simulated with the Euler method, so it's *close enough*, not exact — the tests use a tolerance.

## Tips If You're Stuck

1. **Lander never lands (loops forever or hits the step cap)?** Your thrust is probably *bigger* than gravity, so it floats back up. Use a thrust below 1.62.
2. **Touchdown speed positive?** That means it was moving *up* when the loop ended — again, thrust too strong.
3. **`lander_step` order:** velocity first, then height using the new velocity.
4. **`None` returned?** Don't forget the `return` lines (Exercise 1 returns a list `[h2, v2]`).
