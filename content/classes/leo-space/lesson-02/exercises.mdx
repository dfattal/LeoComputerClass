## Tips If You're Stuck

1. **Distance not staying constant for circular speed?** Make sure you update *both* velocity parts before *both* position parts.
2. **Satellite flies off instantly?** Check the minus signs in `ax`/`ay` — gravity pulls *toward* the center, not away.
3. **`r**3` vs `r**2`?** The 2D gravity uses `r**3` on the bottom (one extra `r` aims the arrow). The plain strength `G*M/r**2` is for the 1D version.
4. **`None` returned?** Don't forget the `return` lines.
